# Действия:

### Выполним команду, которая инициализирует проект миграций с использованием Alembic для асинхронного взаимодействия с базой данных

cd app

alembic init -t async migration

### Создадим базу, если она отсутствует
cd ../

alembic revision --autogenerate -m "Initial revision"

### Создадим таблицы

alembic upgrade head
